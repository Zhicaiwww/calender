# Tech Events Calendar Sync Project (发布会)

## 🎯 Project Goal
Automatically fetch and sync major tech company (Apple, Meta, NVIDIA, etc.) event dates to Apple Calendar.

## 🏗️ Technical Stack
- Python (for scraping)
- standard iCal (.ics) generation
- AppleScript (for local calendar injection)
